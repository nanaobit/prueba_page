Sistema Solar — animado (HTML + CSS)

Esta demostración usa solo HTML y CSS. Las órbitas son animaciones CSS aplicadas a contenedores rotatorios. Pase el ratón por encima para pausar.

Leyenda y notas

  • Escala no es real: tamaños y distancias están estilizados para verse bien en pantalla.
  • Puedes pausar moviendo el cursor sobre el área principal.
  • Si usa prefers-reduced-motion, las animaciones se desactivarán automáticamente.

Atajos

Copie el archivo HTML y ábralo en el navegador. Puede ajustar las variables CSS al inicio para cambiar tamaños y velocidades.
